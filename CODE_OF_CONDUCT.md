# Code of Conduct

## English

Please keep discussion technical, respectful, and evidence-driven.

- Be precise.
- Be constructive.
- Critique code and claims, not people.
- Prefer reproducible bug reports over rhetorical disagreement.

## 中文

请保持讨论技术化、尊重且基于证据。

- 表达准确
- 反馈建设性
- 评价代码与结论，不攻击个人
- 优先提交可复现的问题，而不是情绪化争论
