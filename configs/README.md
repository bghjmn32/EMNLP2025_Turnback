# Configs

This directory contains the public similarity configurations used by the released CLI.

- `similarity.paper.json`: default local similarity setting
- `similarity.loose.json`: more permissive alternative for ablation or debugging

本目录包含公开 CLI 使用的相似度配置。

- `similarity.paper.json`：默认本地相似度配置
- `similarity.loose.json`：更宽松的调试或消融配置
