# Configs

## English

This directory contains public configuration files.

- `similarity.paper.json`: the default local similarity configuration
- `similarity.loose.json`: a more permissive public alternative
- `reproduction.first12.json` and `reproduction.first24.json`: retained for CLI compatibility and legacy benchmarking utilities, but not the center of this public release

## 中文

该目录包含公开配置文件。

- `similarity.paper.json`：默认本地相似度配置
- `similarity.loose.json`：较宽松的公开备选配置
- `reproduction.first12.json` 与 `reproduction.first24.json`：为了 CLI 兼容性和旧基准工具而保留，但不是这次公开发布的核心
