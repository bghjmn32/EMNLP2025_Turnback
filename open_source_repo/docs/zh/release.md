# 中文发布政策

## 这个发布包主要服务什么

这个发布包主要服务一条清晰的公开路线工作流：

1. 原始路线目录
2. 路线生成
3. 反转指令 API 调用
4. Path Builder 执行
5. 本地评分

## 它不追求什么

这个发布包并不追求把所有内部实验产物一股脑丢进公开仓库。这个选择是刻意的。

我们希望公开面保持清晰，原因是：

- 用户能更快看懂仓库
- 关键代码路径不依赖隐藏服务
- 仓库能作为真正长期维护的公开代码库存在

## 为什么仍然保留少量兼容性文件

部分配置和模块被保留，是为了维持 CLI 兼容性并避免破坏已经完成的重构路径。这并不意味着它们是这次公开发布的主线内容。
