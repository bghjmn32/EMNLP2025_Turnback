# 中文 FAQ

## 仓库里包含模型输出吗？

不包含。仓库提供的是路线语料、路线生成代码、prompt / provider 适配、Path Builder 和本地评分。模型调用由用户自己完成。

## 仓库里包含 `data_set/` 吗？

不包含。这个公开发布目录刻意围绕 `36kroutes/` 以及与之配套的代码栈来组织。

## 为什么有些路线文件夹没有完整填充？

因为发布包明确保留了原始语料状态，而不是在打包时静默修改目录结构。

## 所有流程都需要 OpenRouteService 吗？

不需要。只有生成新路线时才需要 ORS。查看已有路线、调用大模型、执行指令和评分都不依赖 ORS。

## 所有流程都需要 OpenAI 或 Gemini 吗？

不需要。只有在你要生成反转指令时，才需要外部大模型 API。
