## Release Principles

- keep the public surface small
- make the route workflow obvious
- avoid hidden dependencies
- separate documentation from experimentation noise

## 发布原则

- 公开面尽量小
- 让路线工作流一眼能懂
- 避免隐藏依赖
- 把文档与实验噪音分开
