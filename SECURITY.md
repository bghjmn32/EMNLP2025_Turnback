# Security Policy

## English

This repository is a research-code release, not a hosted production service.

If you find a security-sensitive issue:

1. do not post secrets, tokens, or exploit details in a public issue
2. contact the maintainers privately first
3. provide a minimal reproduction if possible

Typical public issues in this repository are about:

- malformed route files
- CLI crashes
- incorrect parsing or execution
- packaging mistakes

## 中文

本仓库是研究代码发布，不是托管型线上服务。

如果你发现了安全敏感问题：

1. 不要在公开 issue 中贴出密钥、token 或可直接利用的细节
2. 先私下联系维护者
3. 如果可以，请提供最小复现方式

本仓库更常见的公开问题通常是：

- 路线文件格式异常
- CLI 崩溃
- 解析或执行错误
- 打包问题
